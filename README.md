# ✦ Aura Browser

An immersive, distraction-free, completely fullscreen desktop web browser with built-in privacy protection and custom spotlight-style search navigation.

---

## Key Features

- 🖥️ **Absolute Fullscreen**: Completely borderless interface with no close, minimize, or title bars (exit using `Alt + F4`).
- 🛡️ **Built-in Ad Blocker**: Out-of-the-box tracking and ad-request prevention for faster, cleaner surfing.
- 🔍 **Spotlight HUD Search Overlay**: Toggle a beautiful glassmorphic search dashboard from anywhere with `Ctrl + T`.
- ⚡ **Search Shortcuts (Bangs)**: Search directly on your favorite sites:
  - `!g query` (Google)
  - `!d query` (DuckDuckGo)
  - `!b query` (Bing)
  - `!w query` (Wikipedia)
  - `!yt query` (YouTube)
- ⭐️ **Bookmarks & History**: Press `Ctrl + D` to bookmark any page, and browse your recent history directly in the search overlay suggestion list.

---

## Keyboard Shortcuts Quick Sheet

| Action | Shortcut |
| :--- | :--- |
| **Open Search HUD / Enter URL** | `Ctrl + T` |
| **Add Page to Bookmarks** | `Ctrl + D` |
| **Reload Webpage** | `Ctrl + R` |
| **Navigate History (Back / Forward)** | `Alt + LeftArrow` / `Alt + RightArrow` |
| **Close Browser Application** | `Alt + F4` |

## Installation & Running

### Windows Installer
You can download the latest installer from the **[Releases](https://github.com/your-username/aura-browser/releases)** section:
1. Download **`AuraBrowser Setup 1.0.0.exe`**.
2. Run the installer to setup and launch Aura Browser.
3. *(Alternative)* Download **`AuraBrowser 1.0.0.exe`** for a standalone portable version that runs instantly without installation.

### Run From Source Code
If you want to run or modify the code locally:
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/your-username/aura-browser.git
   ```
3. Navigate to the folder and install dependencies:
   ```bash
   cd aura-browser
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
